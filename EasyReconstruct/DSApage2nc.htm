Decay Search Step #2.2

Extra-track search (Step 2.4 of the reference document)

Do for me
Notice: this step applies to 0μ events with 3 prongs or fewer; if this is not your case, go back
Apply the following steps to search for extra tracks upstream of the main vertex:
  1. Select "Tracks" on the main panel
  2. Write "3" in the "Min Segments" edit box
  3. Browse the Main Vertex and go to the "Neighborhood" tab
  4. Click "Quick Upstream Parents"
  5. Click "Quick Downstream Decays"
  6. Optional: the "quick" selection implements cuts that are looser than those used in the official procedure; however, you can tighten them after first survey by tuning Radius and DeltaZ
For each new track that has been displayed, apply the following steps to validate/discard it:
  1. Open the TrackBrowser for the track.
  2. Go to the "Neighborhood" tab.
  3. Check "Upstream" both for "End" and "Direction".
  4. Ensure that Opening=0.05, Radius=100.0, DeltaSlope=0.05, DeltaZ=100000.0, Max Misses=3.
  5. Click on "Scanback/Scanforth": this performs "Virtual Scanback".
  6. If only one segment appears, then the track stops there; if more segments appear, accept the request to include the segment to the track, and then look carefully at them (going to the "Segments" tab) to determine whether they are really part of the track or not; this step saves manual checks at the microscope. If the track extends more than 3 plates upstream of the vertex, it is not to be checked any more. If it has an upstream vertex, open the VertexBrowser for that vertex and click on "Kill" on the "General" tab. Repeat for the next track (you can use the "Highlight" function to keep memory of checked tracks).
  7. Go to the "Feedback" tab and provisionally add the track to feedback.
  8. If the additional track has enough segments, EasyReconstruct will automatically compute the RMSδθtransv and RMSδθlong: go to the "General" tab to look at them. If either RMS exceeds 0.020, go to the "Feedback" tab and mark the track as "Low Momentum" in the Decay Search dropdown list; such tracks are not to be checked any more.
  9. If the track survives all these filters, it is to be visually inspected to check it is not an artifact of reconstruction. Go to the "Projection" tab and append segment search lines to the file for manual checks; it is recommended to check the track at least in the most upstream plate where it has been found, and in the other upstream plates, up to one or two plates before the vertex.
Notice that the output file is always appended and never overwritten.
