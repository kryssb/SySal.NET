Decay Search Step #4

Use the results of manual checks (contained in Steps 1,2,3 of the reference document)

Following these steps to fully exploit the information of manual checks:
  1. In the main panel, select "Tracks", type event in the "With attribute" edit box and click "Plot"
  2. Empty the "With attribute" edit box, deselect all datasets and select only the MANCHECK data sets; click "Plot"
  3. For each resolved pair of tracks, browse each track, go to the "Feedback" tab and set the particle type for both tracks as "ELECTRON"; go to the main panel, create a new vertex fit and add the two tracks of the pair; include the vertex in the feedback ("Feedback" tab in the VertexBrowser). This automatically disconnects the pair from the original vertex.
  4. For each single reconstructed track that has been found to be an e+e- pair, go to the "Feedback" tab and set the particle type as "EPAIR"; click on "Add a single-prong vertex at Z" = -1 (notice the default is -500); include the upstream vertex to the feedback. This automatically disconnects the pair from the original vertex.
  5. Browse each track that is not confirmed by manual checks, go to the "Feedback" tab and remove the track from feedback.
  6. If disproved tracks are attached to a vertex, you need to disconnect them. To do this, go to the main panel and create a new vertex fit; then, on the VertexBrowser, go to the "Vertex Fit" tab and add all vertex tracks to the fit; go to the "General" tab and click on the "Kill" button; then remove the disproved tracks from the vertex fit; last, convert this to a vertex and include it in the feedback (on the "Feedback" tab); do not forget to flag the "Primary vertex" as such if it is deleted and recreated.
