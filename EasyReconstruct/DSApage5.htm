Decay Search Step #5

Refining the definition of the Primary Vertex and Extra-tracks; In-track decay search (Step 2.3 of the reference document)

Clear the plot, type "event" in the "With Attribute" box of the main panel, select "Tracks" and click "Plot".
For each track shown, do the following steps:
  1. If the track is an isolated muon or hadron track, believed to come from the primary interaction, open the TrackBrowser and go to the "Feedback" tab and create an upstream vertex; if there are tracks downstream of it, put the vertex at Z = -1000; otherwise put the vertex at Z = -500;
  2. On the TrackBrowser form for the track, and go to the "General" tab. If RMSδθtransv>0.020 or RMSδθlong>0.020, go to the "Feedback" tab and flag the track as "LowMomentum"; go to the next track.
  3. If the Kink/Layer box is orange, the track is suspected to hide a kink; in this case, go to the "Feedback" tab and flag the track with "Scanforth to be done"; go to the "Projection" tab, define a plate from which you want to start the Scanforth, and append the corresponding projection to the file for Scanforth initialization; go to the next track.
  4. If the track is attached to the primary vertex, go to the "Feedback" tab and flag it as "PrimaryVertexTrack"; also fill other information about the track. If you apply systematic Scanforth, go to the "Projection" tab, define a plate from which you want to start the Scanforth, and append the corresponding projection to the file for Scanforth initialization; go to the next track.
  5. For each track that is not attached to the primary vertex, go to the "Feedback" tab, fill the relevant information, and click on "Add single-prong vertex"; use Z = -1000 if the track is upstream of the primary vertex, -1 if it is downstream of it.
