Once you got possible primary tracks, either they are attached to the primary vertex (upstream) or not. If you believe that there is at least one track that has not been associated to the vertex, but should be attached, do the following steps:
  1. On the main panel, type a name for a new vertex fit, and click "Vertex Fit"
  2. Add tracks to the vertex fit, using Weight=1; this can be done from the TrackBrowser "Vertex Fit" tab, and from the VertexBrowser "Vertex Fit" tab; the advantage of the latter is that all the tracks attached to an automatically reconstructed vertex are attached to the vertex fit, so it is not required to keep memory of which tracks have already been associated; a useful trick is to highlight tracks that have already been added (the "Highlight" checkbox is always present in the TrackBrowser
  3. On the vertex fit panel, click on "Convert to Vertex" to make a new vertex with these tracks.
  4. If there is only one prong, one still defines a fictitious vertex halfway in the lead plate that is immediately upstream: browse its track, open the "Feedback" tab and click on "Add single-prong vertex at Z"=-500
  5. Browse the new vertex, go to the "Feedback" tab on the VertexBrowser and check "Include vertex in feedback" and "Primary Vertex".
  6. Open the "Neighborhood" tab and click on "Quick Downstream Tracks"
  7. Optional: without changing the settings, click on "Related Segments" and look for aligned segments in the lab reconstruction!
  8. Create a new vertex fit and repeat the procedure if you believe that the vertex can be improved
If you are in doubt whether a track is attached or not to the vertex, attach it: later you will be able to decide after manual checks.
Warning: repeatedly creating new vertex fits and converting them to vertices will leave several invalid vertices (i.e. empty or with one track). These vertices will be discarded when the file is saved, but vertex numbering will change as a consequence.
The main vertex found should be flagged: on its VertexBrowser, go to the "Feedback" tab and check "Include vertex in feedback" and "Primary Vertex".
Manual checks are a useful source of information about the real topology of the event, and are usually needed to confirm/discard tracks with high slope. In order to have a list of the tracks related to the vertex to be visually inspected, open the "Projection" tab in the VertexBrowser and define the file where you want to append lines, then click on "Append to". If Downstream tracks is selected, for each track downstream of the vertex the following segment search lines will be added to manual checks:
  • one segment at each plate between the track end and the vertex point;
  • the segments at the last plates where a track has been seen, which is useful to check e+e-pairs, and to confirm that the track is not a fake; the number of plates checked is defined by track layers;
  • one segment search for each plate upstream of the vertex, to check track disappearance; the number of plates upstream is defined by stop layers.
If Upstream tracks is selected, the same applies to upstream tracks, swapping downstream and upstream directions.
Notice that the output file is always appended and never overwritten.
