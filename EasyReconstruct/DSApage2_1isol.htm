Decay Search Step #2.3

Decay search for isolated hadron and muon tracks (Step 3 of the reference document)

Notice: this step applies to 1μ and 0μ events with isolated muon or hadron; if this is not your case, go back
Apply the following steps to search for parent tracks of the isolated track:
  1. Select "Segments" on the main panel
  2. Write N >= 20 || (S < 0 && N >= 12) in the "Filter Vars" edit box: this will discard bad base tracks and very weak microtracks.
  3. Make sure that Opening=-0.05, Radius=500.0, DeltaSlope=-0.05, DeltaZ=3900.0, IP=20.0
  4. Select Upstream and click "Related segments"
  5. Make sure that Opening=-0.05, Radius=500.0, DeltaSlope=-0.05, DeltaZ=5200.0, IP=20.0
  6. Select Downstream and click "Related segments"
For each new segment that has been displayed, apply the following steps to validate/discard it:
  1. Click it and check the track it belongs to; if Track = -1, then promote the segment to track.
  2. Go to the "Neighborhood" tab.
  3. Check "Upstream" both for "End" and "Direction".
  4. Ensure that Opening=0.05, Radius=100.0, DeltaSlope=0.05, DeltaZ=100000.0, Max Misses=3.
  5. Click on "Scanback/Scanforth": this performs "Virtual Scanback".
  6. check "Downstream" both for "End" and "Direction" and click on "Scanback/Scanforth": this performs "Virtual Scanforth".
Notice that the output file is always appended and never overwritten.
