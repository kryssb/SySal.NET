Decay Search Step #6

Short Decay search (Step 2.1 of the reference document)

If one or more tracks attached to the primary vertex have IP exceeding 10 μm, take the following steps to strenghten/disprove this signal:
  1. In the main panel, create a new empty vertex fit.
  2. Open the VertexBrowser for the vertex.
  3. Go to the "Vertex Fit" tab.
  4. Select "Use last segment".
  5. Ensure that Extrapolation Distance = 3900.
  6. Select the empty vertex fit and click on "Add to fit".
  7. Remove from the vertex fit the tracks whose IP exceeds 10 μm.
  8. Open the TrackBrowser for each track whose IP exceeded 10 μm, go to the "Vertex Fit" tab, set Weight = 1e-6, enable "Use last segment", ensure ExtrapolationDistance=3900, choose the vertex fit and then click on "Upstream".
  9. Select the Upstream vertex ("General" tab) from any of the primary vertex tracks
  10. Click on the "Kill" button.
  11. Convert the new vertex fit to a vertex; go to the "Feedabck" tab and set the "Feedback" and "Primary" flags as needed.
Now the vertex has been recomputed using only tracks with IP < 10 μm, but all tracks are associated.
