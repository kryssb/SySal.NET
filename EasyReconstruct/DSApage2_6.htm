Decay Search Step #2.6

Search for segments connecting vertices or tracks (contained in Steps 2,3 of the reference document)

This step allows you to perform visual inspection to look for undetected micro/basetracks connecting two vertices or a vertex and a track, or two tracks.
  1. For each track that has no creation vertex, add a vertex by opening its TrackBrowser, going to the "Feedback" tab and clicking on "Add single-prong vertex at Z", using the default value of 500 μm.
  2. For each pair of vertices (real or fictitious) that might be connected by undetected micro/basetracks, open the VertexBrowser of one of the two and go to the "Projections" tab.
  3. Deselect "Upstream tracks" and "Downstream tracks", select "Tracks connecting to vertex" and enter the id of the other vertex of the pair.
  4. Select the file where you want to put the segment search lines.
  5. Click on "Append To".
Notice that the file with the results of the manual checks will now be ordered by plate.
